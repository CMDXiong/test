Mouse menu
Overview
The mouse-menu provides a quick call to some frequently used functions. To popup the mouse-menu, right click with your mouse anywhere in the canvas.

Zoom back
When you zoomed in on the picture, use this command to zoom back. Everytime it is clicked, it zooms back one step. This is possible until the zoomfactor is 1. The Zoom back option becomes unavaible then.

zoomback

Zoom out
To view the complete drawing again, after zooming in, click Zoom out. This option is unavaible when the zoomfactor is 1.

zoomout

View structure
The View structure option is also available in the View menu. Please go there for an explanation.

View layers
The View layers option is also available in the View menu. Please go there for an explanation.

Set relative start
Set relative start defines the first point that is used for calculating distances. The second point can be set with the next item in the mouse-menu: Calculate distance.

The starting point is set at (0,0) when the program is started, and when a new file is loaded.

startpoint

Calculate distance
This item calculates the absolute distance, the distance in x, the distance in y and the angle from the start point to the current point.
